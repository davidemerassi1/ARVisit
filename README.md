# ARVisit

Questa è la repository dell'app iOS **ARVisit**, sviluppata come progetto per il corso di Sviluppo di applicazioni mobili. 
L'app permette di eseguire visite museali accessibili sfruttando la realtà aumentata **in ambienti già esistenti**.

## Repository di creazione contenuti

La parte dedicata alla **creazione di nuovi ambienti** si trova in questa repository separata:  
[ARVisit Creator](<https://github.com/davidemerassi1/ARVisit-Creator>)
