# ARVisit

Progetto universitario sviluppato per il corso di Sviluppo di applicazioni mobili.

**ARVisit** è un’applicazione iOS che permette di eseguire **visite museali accessibili in realtà aumentata** utilizzando il proprio iPhone, fornendo informazioni contestuali e segnalazioni utili direttamente nell’ambiente reale.

Il progetto è composto da due applicazioni separate:
- **ARVisit** (questa app), utilizzata dai visitatori
- [**ARVisit Creator**](https://github.com/davidemerassi1/ARVisit-Creator), utilizzata per la creazione dei contenuti

## Descrizione

ARVisit consente ai visitatori di utilizzare i contenuti creati con *ARVisit Creator* per svolgere una visita museale in realtà aumentata.  
Durante la visita è possibile visualizzare:
- punti di interesse (opere)
- punti di servizio (bagni, bar, ecc.)
- punti potenzialmente pericolosi (gradini, ostacoli, ecc.)

Tutti gli elementi vengono mostrati direttamente nello spazio reale attraverso la realtà aumentata.

## Tecnologie utilizzate

- **iOS**
- **ARKit** (framework Apple per la realtà aumentata)
